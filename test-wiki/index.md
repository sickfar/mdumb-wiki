# Welcome to MDumb Wiki

This is a simple, file-based wiki system built with Nuxt 4. It reads markdown files from a directory and renders them as a navigable wiki.

## Features

- File-based content management
- Markdown support with syntax highlighting
- Hot reload during development
- Hierarchical navigation
- Full-text search

## Getting Started

Browse the documentation using the navigation menu.
